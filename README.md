熱中症対策ナビ（Heat Safety Navi）

熱中症対策ナビは、
熱中症の予防・判断・学習・緊急対応を支援するWeb / PWAアプリです。

スマートフォンやPCから利用でき、
ホーム画面に追加することでアプリのように使用できます。

プロジェクト目的

近年、夏季の気温上昇により熱中症のリスクが高まっています。
特に学生の部活動や通学時など、日常生活の中で適切な判断が求められます。

本プロジェクトは熱中症リスクの判断支援・正しい予防・行動の提示学習機能の提供・緊急時の対応支援を目的として開発されています。

主な機能
1. 119通報確認サイト
熱中症症状を確認し、119通報が必要か判断を支援します。
例：意識障害・高体温・応急処置方法

2. 熱リポ
利用者から熱中症発生情報・暑さ体感などを投稿し、地域情報として共有します。

3. 熱中症対策ナビ紹介
アプリの
目的・使用方法・熱中症基礎知識を説明します。

4. やっちゃダメリスト
熱中症の原因になる行動を表示します。

例：水分補給不足・長時間の直射日光・休憩なしの運動

5. 熱中症シミュレーション
状況を選択して・熱中症リスク・正しい行動を疑似体験できます。

6. 熱中症クイズ

クイズ形式で熱中症対策を学習できます。

7. 天気予報

現在の気温・湿度・暑さ指数・危険度を表示します。

8. 熱中症予防
日常生活でできる予防方法を表示します。
